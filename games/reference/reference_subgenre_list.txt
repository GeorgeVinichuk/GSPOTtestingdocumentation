GET reference subgenre list
===

Действия в Postman
--
1. Создать запрос

- метод - GET

- URL - games.alpha.g-spot.website/api/v1/reference/subgenre/

2. Вкладка Authorization

Авторизироваться: используя валидные значения Username:tester, Password:tester


4. Отправить запрос

Ожидаемый результат
--

1. Status ответа: 200 OK.

2. В Body ответа вернулся список поджанров. Пример:

```
{
count*	integer
next	string($uri)
x-nullable: true
previous	string($uri)
x-nullable: true
results*	[SubGenre{
id	integer
title: ID
readOnly: true
name*	string
title: Subgenre
maxLength: 50
minLength: 1
 
}]
 
}
```
